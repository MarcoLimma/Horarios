Buscamos ajudar na melhoria da qualidade do serviço de transporte público na cidade de Petrópolis. Mesmo que nossa parte seja ínfima, acreditamos que a contribuição de cada um é sempre de grande valor para a constante evolução e melhoria da sociedade.
Este projeto é de iniciativa voluntária e foi desenvolvido sem nenhum tipo de incentivo financeiro.

Apenas tecnologias open source foram utilizadas no seu desenvolvimento, agradecemos a elas:

  • MongoDB - Banco de dados NoSql
  • Express - Framework web para Node.js
  • AngularJS - Framework JavaScript client-side
  • Node.js - Interpretador JavaScript server-side
